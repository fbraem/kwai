# Command Line Interface
A command line interface (CLI) is provided to help managing the kwai system. To run the cli, use the kwai_cli.py 
script:

`kwai_cli.py --help`

::: kwai.cli.bus

        show_root_full_path: False
        show_signature: False
        members:
            - show
            - test
